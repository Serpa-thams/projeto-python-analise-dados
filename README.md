# 📊 Análise de Dados Financeiros com Python

## 📌 Objetivo
Analisar dados reais de fluxo de caixa de um escritório de advocacia utilizando Python, com foco na organização, limpeza e análise de dados financeiros.

## 🧠 Contexto
Os dados estavam originalmente organizados em planilhas, com inconsistências e duplicidades (como pagamentos de fatura de cartão). O projeto simula a estruturação e análise desses dados de forma mais confiável e escalável.

## 🛠 Tecnologias
- Python
- pandas
- Excel

## ⚙️ Etapas realizadas
- Leitura de dados a partir de Excel
- Limpeza e padronização de dados
- Tratamento de inconsistências (remoção de duplicidades)
- Criação de coluna de valor (receita/despesa)
- Análise de receitas vs despesas
- Agrupamento por categoria
- Identificação de principais gastos e receitas

## 📊 Resultados
- Identificação de prejuízo no período analisado
- Mapeamento das principais categorias de despesa
- Destaque das maiores fontes de receita
- Estruturação dos dados para tomada de decisão
